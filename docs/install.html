Install on iPhone

Use a configuration profile to add View Caster to your home screen.

Option A — Profile (recommended)

On your iPhone, tap the button below. iOS will ask to review and install a profile, then add View Caster to your home screen.

Install View Caster Profile
  1. Tap Install View Caster Profile above
  2. Tap Allow when iOS asks to download the profile
  3. Open Settings → Profile Downloaded (or General → VPN & Device Management)
  4. Tap Install, enter passcode, confirm
  5. Open the new View Caster icon on your home screen

The profile is unsigned (normal for home-brew profiles). iOS will show “Not Verified” — tap Install anyway. It only adds a home-screen shortcut to the relay page; it does not change system settings.

Option B — Native app (best camera)

For the full relay app with native camera, download the IPA and sideload with Sideloadly on a PC.

Download IPA (GitHub Releases)

iOS cannot install unsigned IPAs from a profile link alone — Sideloadly re-signs with your Apple ID. The profile (Option A) works without a computer but uses the web relay.

After install: open View Caster on your phone → wait for the green Relay online dot and a 6-digit code → tap Allow camera permissions → tap Connect Meta AI glasses to copy the glasses URL → enter the code on desktop and glasses.

If you see “Relay offline”, force-quit the app and reopen it, or tap Restart relay. No signaling server deploy is required for the profile app.
