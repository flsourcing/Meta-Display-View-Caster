Install on iPhone

Use a configuration profile to add View Caster to your home screen.

Option A — Profile (phone camera)

Quick install — no PC after setup. Uses your iPhone camera for the live stream (not glasses POV).

Install View Caster Profile
  1. Tap Install View Caster Profile above
  2. Tap Allow when iOS asks to download the profile
  3. Open Settings → Profile Downloaded (or General → VPN & Device Management)
  4. Tap Install, enter passcode, confirm
  5. Open the new View Caster icon on your home screen

The profile is unsigned (normal for home-brew profiles). iOS will show “Not Verified” — tap Install anyway. It only adds a home-screen shortcut to the relay page; it does not change system settings.

Option B — Native app (glasses camera)

Streams from Meta glasses camera (not iPhone). Opens Meta AI for app connection + camera permission. Requires sideloading.

Download IPA (GitHub Releases)
  1. Download ViewCasterRelay-unsigned.ipa
  2. Patch Team ID (required for Meta AI connection):
    cd ios
    .\patch-ipa-teamid.ps1 -TeamId YOUR10CHARID -Ipa path\to\ViewCasterRelay-unsigned.ipa
    Use the Team ID Sideloadly shows when installing, or the value shown in the app after first open.
  3. Install the -patched.ipa with Sideloadly (same Apple ID as iPhone)
  4. Trust: Settings → General → VPN & Device Management → Trust
  5. Meta AI → Settings → App Info → tap version 5× → enable Developer Mode
  6. Open relay app → Start relay (wait up to 60s for code) → Connect Meta AI → approve → Allow glasses camera

Requires iOS 17+. Step 2 is required — without Team ID in the IPA, Meta AI opens but won’t show the app connection prompt.

After profile install: Connect Meta AI → add glasses web app → allow phone camera → enter code on desktop & glasses → Live Stream.

Deploy the signaling server once: deploy-server.html
